
Informações para a aula
- Uma variavel é um container que armazena dado
- Uma variavel se comporta como se fosse o valor que a contem

Passos para a variavel:
- Declaração (var, let, const)
- Atribuição (operador de =)

Observações: 
- Variaveis que são declaradas SEM 'let' só funcionam globalmente (em todo o código)
- Variaveis que são declaradas COM 'let' só funcionam dentro do bloco que foram criadas
- As variaveis 'getElementById' alteram uma tag que possui tal ID para imprimir diretamente na página HTML.

Sintaxe:
- document.getElementById("IdQualquer").innerHTML - é uma forma de acessar o conteúdo HTML dentro de um elemento específico na página, identificado pelo seu ID, e também de modificar esse conteúdo.
    ex: document.getElementById("IdQualquer").innerHTML = "Novo texto aqui!" > Altera o conteúdo do elemento com o ID "IdQualquer" para exibir "Novo texto aqui!"
