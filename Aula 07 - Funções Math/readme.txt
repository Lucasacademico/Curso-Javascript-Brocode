Aula 1
- Criar folder onde as aulas ficaram salvas
- Abrir o folder criado no VScode
- Criar os arquivos index.html, index.js, e style.css no folder

Após preparar a estrutura e linkar arquivos CSS e JS no HTML:
- Abriremos o arquivo html via LiveServer
- Inspecionaremos a página aberta, e clicaremos em console
Pronto, aqui será o ambiente que veremos a maioria dos códigos JS em funcionamento  

Sintaxes:
- console.log() - Imprime os valores dentro do parentesis
    ex: console.log("Lucas") - Será printado no log > Lucas

- window.alert(); - Imprime os valores dentro do parentesis em uma janela quando o site é aberto/recarregado
    ex: window.alert("Eu realmente amo pizza"); - Será printado no log > Eu realmente amo pizza
